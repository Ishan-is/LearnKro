# Enable Course Progress Tracking

Currently, when a student enrolls in a course, they are added to the course's student list, but we lack a dedicated system to track their individual learning progress (e.g., which lectures they've watched, their completion percentage). Because of this missing backend structure, the "Continue Learning" dashboard throws a "Course Not Found" error.

## Proposed Changes

### 1. Create Enrollment Model
I will create a new MongoDB model (`Enrollment.js`) that tracks a student's specific progress through a course. It will store:
- The `user` and `course` references.
- `currentLecture`: The index of the lecture the student is currently watching.
- `completedLectures`: An array of lecture indexes they have finished.
- `completionPercentage`: A number from 0 to 100.
- `isCompleted`: A boolean flag indicating if they've finished the whole course.
