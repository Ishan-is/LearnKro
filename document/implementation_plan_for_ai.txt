# Enable AI Quiz Generator

The frontend for the AI Quiz Generator is already beautifully designed, but it currently hits a "Route Not Found" error because the backend logic to actually connect to the AI and generate the quiz does not exist. 

This plan outlines the steps to build the `Quiz` model and the backend routes using the Groq AI SDK.

## Proposed Changes

### 1. Create Quiz Database Model
I will create a MongoDB model to store generated quizzes and their results.
- **Fields:** `user` (student ID), `topic`, `difficulty`, `timeLimit`, `totalQuestions`, `title`, and an array of `questions`.
- **Question Schema:** `question`, `options` (array of strings), `correctAnswer` (index), `explanation`, `difficulty`.
- **Results:** `score`, `percentage`, `passed` (boolean), `isCompleted` (boolean).

### 2. Implement Groq AI Quiz Generation Logic
I will create the backend routes to handle the quiz lifecycle and prompt the Groq AI model.
- **POST `/generate`**: 
  - Connects to the Groq AI API (`groq-sdk`).
  - Instructs the AI to generate a quiz based on the user's `topic`, `difficulty`, and `totalQuestions`.
  - Parses the AI's JSON response and saves the newly generated `Quiz` to MongoDB.
- **GET `/:quizId`**: Fetches the quiz for the frontend to display, purposefully hiding the `correctAnswer` and `explanation` from the payload so students can't cheat!
- **POST `/:quizId/submit`**: Grades the student's submitted array of answers against the database, calculates the score/percentage, and returns the full results (including explanations).
